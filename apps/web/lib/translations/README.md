## Translation Guidelines

When creating translations:
1. Maintain marketing tone (professional, friendly, trustworthy)
2. Preserve brand name "Uprent" 
3. Keep technical terms accurate (AI, 15 seconds, 1,500+ sources)
4. Use culturally appropriate phrasing
5. Ensure SEO-friendly text

## Adding New Translations

1. Copy en.ts structure
2. Translate all strings maintaining the same structure
3. Import in index.ts
4. Add to translations object
